欢迎来到 GinaWai

这是我博客的第一篇文章。

用 Hugo 搭建静态博客，托管在 GitHub Pages，通过 Cloudflare 加速，使用自定义域名 ginawai.com。

为什么选择自托管博客？

  • 完全掌控内容：不依赖任何第三方平台
  • 免费托管：GitHub Pages 完全免费
  • 速度极快：静态页面 + Cloudflare CDN
  • Markdown 写作：专注内容，格式简洁
